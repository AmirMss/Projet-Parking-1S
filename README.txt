# Projet-Parking-1S
Bonjour 🤗🤗🤗;

Pour lancer le programme , vous devez au préalable avoir installer ( sox , screen , pkill ).
make - pour makefile le programme 
Pour pouvoir lancer le jeu vous devez choisir obligatoirement une map './Parking files/01.map' .
Une fois le jeu lancer vous devez choisir un mode de jeu (attention aux oreilles) .
Une fois le mode de jeu lancé pour l'arrêter, il faut utiliser les touches'CTRL+C' une fenêtre de dialogue va alors s'ouvrir
vous allez devoir choisir entre Y ou y et n ou N.
n ou N - Vous reprenez la partie la ou vous vous êtes arrêté.
y ou Y - Vous quitter le jeu .

Bon jeu à vous🍻🍻 .
